AMuPDF is an document viewer for Android based on the MuPDF code base.
支持以下格式/Supported the file in following formats:
  • PDF
  • CBZ
  • EPUB
  • XPS (OpenXPS)

特性/Supported features:
  • 图书目录
  • 历史记录
  • 搜索图书
  • 收藏列表
  • 多点触摸手势
  • 流畅快速滑动阅读,迅速加载大pdf文件
  • 文本重排,可显示图片,设置字体与颜色
  • 自动切边
  • 保留MuPDF阅读器
