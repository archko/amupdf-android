Third-party software used by this program:
  • MuPDF a lightweight PDF and XPS viewer
  • EBookDroid a PDF viewer
  • Djvulibre a djvu viewer
  • Recyclerview
  • Material
  • k2pdfopt K2pdfopt optimizes PDF/DJVU files for mobile e-readers (e.g. the Kindle) and smartphones.
  • Subsampling-scale-image-view a image viewer
  • MMKV MMKV is an efficient, small, easy-to-use mobile key-value storage framework
  • Coil Coil is a Kotlin-based image loading library that uses Coroutines, OkHttp, and AndroidX Lifecycles
  • Icon Material Design Icons
