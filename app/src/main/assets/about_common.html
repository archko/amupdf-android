Dragon Viewer is an document/image/txt/pdf/epub/mobi viewer for Android based on the MuPDF
issue:https://github.com/archko/amupdf-android/issues
支持以下格式/Supported the file in following formats:
  • PDF
  • EPUB
  • MOBI
  • AZW3
  • DJVU
  • XPS (OpenXPS)
  • CBZ
  • DOCX,PPTX,XLSX
  • Images(png,jpg,tiff,heic.eg)

Supported features:
  • 文件浏览 / File browser
  • 书库 / Library
  • 历史,收藏 / Histories,Favorites
  • 目录相册 / Album
  • 自动切边 / Auto crop bounds
  • 文本识别 / Ocr for page
  • 超大图浏览 / Big image viewer
  • 多点触摸 / Multitouch
  • 双向滚动 / Horizontal,Vertical scroll
  • 夜晚模式 / Day and Night theme
  • 背景滤镜 / Background filter
  • 转换epub / Convert to Epub
  • 导出图片 / Extract pdf,epub to images/html
  • 创建pdf / Create pdf from images/txt
  • 同步webdav / Sync histories to webdav
