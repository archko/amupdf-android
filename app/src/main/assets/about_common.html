Dragon Viewer is an document/image/txt viewer for Android based on the MuPDF issue(报告错误):https://github.com/archko/amupdf-android/issues
支持以下格式/Supported the file in following formats:
  • PDF
  • EPUB
  • MOBI
  • DJVU
  • XPS (OpenXPS)
  • CBZ
  • Images(png,jpg,tiff,heic.eg)

特性/Supported features:
  • 文件浏览
  • 历史记录
  • 目录相册
  • 自动切边
  • ocr文本识别
  • 超大图浏览
  • 多点触摸手势
  • 双向滚动
  • 夜晚模式
  • 背景滤镜
